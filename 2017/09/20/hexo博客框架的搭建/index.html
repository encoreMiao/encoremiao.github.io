搭建hexo博客框架

hexo:快速简洁且高效的博客框架，hexo使用markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装:
安装前确认电脑中安装node.js和git
(1)安装node.js:
curl https://raw.github.com/creationix/nvm/master/install.sh | sh
nvm install stable
(2)安装Hexo
Nom install -g hexo-cli

在指定文件夹中新建所需要的文件:
hexo init 文件夹
cd 文件夹
npm install

新建文章:
hexo new 博客名称 // 新建的文章都在source/ 的目录下 open之后可以直接编辑
hexo generate // 编辑md后生成html 简写:hexo g
hexo server // 可以简写成ser 本地预览

主题设置:
git clone https://github.com/shinux/hexo-theme-adoubi.git themes/adoubi //主题设置
theme: adoubi // 配置文件中设置主题样式
重新generate和server预览

发布:
在github上创建新仓库
在_config.yml中设置deploy
deploy:
type: github
repository: https://github.com/encoreMiao/mmblog
hexo deploy //简写 hexo d ，注意在部署之前先generate
//npm install hexo-deployer-git –save//hexo 更新到3.0之后，github需要改为git，deploy时出现错误，Deployer not found: git执行该代码

ps:
npm install hexo-deployer-git –save 出现错误
